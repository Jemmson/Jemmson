How it works

This templates purpose is to add a task to the job. It is a modal window that will display the basic information needed to add a task to the job. A task is a piece of work to be performed by the contractor or the sub contractor.

Types of tasks to be added

  1. If the name of the task already exists in the task table and is associated to the contractor that is creating the task then the task should simply be updated and not add new task to the task table
  2. If the name of the task already exists in the task table and is associated to the contractor that is creating the task and the contractor changes any of the details then there should be an option to update the existing or leave the task the same
  3. If the task name is different than a task name in the task table for that contractor then add a new task
